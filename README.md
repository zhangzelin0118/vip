# 第三周作业-员工管理系统

### 作者介绍：
* author：张泽林
* My  Blog:<http://www.cnblogs.com/zhangzelin930118/>

### 功能介绍：
* 查询文件中指定员工的工资，并打印
* 增加新的员工信息，并能查询
* 删除员工信息
* 查询文件中所有员工的姓名，为查询功能提供员工信息
* 退出程序

### 环境依赖：
* Python3.0+

### 模块调用：

- OS

###运行说明：
* 把‘’info.txt‘’文本和‘’员工工资管理系统.py‘文件’放在同一目录下
* 在Linux平台或者Windows下运行‘’员工工资管理系统.py‘文件
* 根据提示完成查询、增加、删除、查询所有员工列表、删除配置等操作。
